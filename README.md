# Palindrome Checker

A simple Python program that checks if a word, sentence, or phrase is a palindrome without using any external libraries.

## 📋 Description

A palindrome is a word, phrase, or sequence that reads the same forwards and backwards (ignoring spaces, punctuation, and capitalization). This program takes user input, reverses it, and determines if it's a palindrome.

Examples of palindromes:
- "radar"
- "race car"
- "A man a plan a canal Panama"
- "never odd or even"

## 🚀 Features

- Checks both single words and multi-word phrases
- Preserves original spacing for sentence input
- Displays original and reversed versions for comparison
- No external imports required
- Simple command-line interface

## 🔧 How It Works

1. **Input**: User enters a word or phrase
2. **Processing**: Input is split into a list and reversed
3. **Comparison**: Original list is compared with reversed list
4. **Output**: Displays whether the input is a palindrome
